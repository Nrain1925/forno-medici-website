FORNO MEDICI WEBSITE PACKAGE

What's included:
- index.html
- styles.css
- script.js
- images/ folder

How to upload to GitHub:
1. Create a new repository named: forno-medici-website
2. Click "Add file" > "Upload files"
3. Drag ALL files and folders from this package into GitHub
4. Scroll down and click "Commit changes"

How to deploy on Vercel:
1. Log in to vercel.com
2. Click "Add New Project"
3. Choose your GitHub repository: forno-medici-website
4. Click Deploy

How to connect your domain:
1. In Vercel, open the project
2. Go to Settings > Domains
3. Add fornomedici.com
4. Follow the DNS instructions shown by Vercel
